## Dynamic, constante e final;

### dynamic
Pode receber qualquer tipo  
Pode mudar de tipo depois  
Desativa a verificação de tipo em tempo de compilação   

### final
Só pode ser atribuído uma única vez  
Valor pode ser definido em tempo de execução (runtime)  
Não pode mudar depois de definido  

### const  
Também só pode ser atribuído uma vez  
Precisa ser definido em tempo de compilação  
O valor é totalmente imutável  

## Funções nomeadas com/sem retorno; com/sem parâmetros


Recebe dados.

```
int somar(int a, int b) {
  return a + b;
}
```

### Função com retorno
Retorna um valor

```
String saudacao() {
  return "Olá!";
}
```

### Função com parâmetros opcional   
colocar o parametro entre colchetes[] permite que nada seja passado, mas isso tem que ser tratado no codigo, nos parametros deve-se colocar os obrigatorios antes dos opcionais.
```
void apresentar(String nome, [int? idade]){
  if(idade != null){
    print("meu nome e $nome e tenho $idade anos");
  }else{
    print("meu nome e $nome");
  }
}
```
OU assim, deixando um atributo padrão 
```
void apresentar(String nome, [int? idade = 20]){
    print("meu nome e $nome e tenho $idade anos");
}
```

### Função com parâmetros nomeados    
colocar o parametro entre chaves[], muda a forma de se chamar a função, passando o argumento usando o nome da variavel, não importando a ordem.

```
void apresentar({String? nome, int? idade, String? curso}) {
  print(
    "Nome: ${nome ?? "Francisco"}, idade: ${idade ?? "20"}, Curso: ${curso ?? "E.S"}",
  );
}

void main() {
  apresentar(idade: 42, nome: "Francisco");
}
```

## Parâmetros opcionais, nomeados e posicionais (padrões e sintaxe)       

### required 
```
void saudacao({required String nome}) {
  print('Olá, $nome!');
}

saudacao(nome: 'João'); // Olá, João!
```
