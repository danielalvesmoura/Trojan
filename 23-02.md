# Planejamento da Apresentação

Optamos por buscar um tema não sugerido e selecionamos o tema:

### Controle de Fluxo

● if / else  
● switch  
● while  
● do while  
● for  
● for in  

## Divisão de Trabalho

Dividimos as partes entre os membros da equipe da seguinte forma:  

| Luiz | Daniel | Francisco |
|:----------|:--------:|----------:|
| if / else|while|for|
| switch|do while|for in|


## Metodologia de Apresentação

● Como IDE, utilizaremos o Visual Studio Code.  
● Faremos um rodízio onde um membro apresenta enquanto o escreve o código descrito pelo apresentador.  
● Cada membro terá de 3 a 4 minutos para apresentar sua parte, totalizando 9-12 minutos de apresentação.

import 'dart:io';

main() {
  //          WHILE

  //         contador

  int i = 0;

  while (i < 5) {
    print(i);
    i++;
  }




  //            repetição sem limite prévio

  int i = 0;

  while (i != 1) {
    i = int.parse(stdin.readLineSync()!);

    if (i != 1) {
      print("não é 1");
    }
  }



  //          cuidado para não criar loops infinitos

  int i = 0;

  while (i < 5) {
    print(i);
  }




  //          break

  int i = 0;

  while (true) {
    print(i);

    if (i == 5) {
      break;
    }

    i++;
  }



  // continue

  int i = 0;

  while (i < 5) {
    i++;

    if (i == 3) {
      continue;
    }

    print(i);
  }



  // do while

  String? resposta;

  do {
    print("Quer continuar? [s] sim [n] não");
    resposta = stdin.readLineSync();
  } while (resposta != "n");
}

import 'dart:io';

void main() {
  List<String> alunos = [
    "ana",
    "alex",
    "bruno",
    "carlos",
    "cleiton",
    "daniel",
    "eduardo",
    "fernanda",
    "gabriel",
    "helena",
    "isabela",
    "joão",
    "karla",
    "lucas",
    "mariana",
    "nicolas",
    "otavio",
    "paula",
    "rafael",
    "sabrina",
    "thiago",
    "victor",
    "yasmin",
  ];
  // ============================== FOR ===================================
  // 1) For com índice: quando você precisa do número (posição)
  // Ex.: lista na UI com numeração / ranking / acessar outro array pelo mesmo índice
  for (int i = 0; i < alunos.length; i++) {
    final nome = alunos[i];

    // 2) continue: pular alguns itens
    if (nome.length <= 4) continue; // pula nomes curtos

    print("${i + 1}. $nome"); // i+1 para mostrar ranking começando em 1

    // 3) break: parar quando achar algo
    if (nome == "Helena") {
      print("Encontrei a Helena! Parando o loop (break).");
      break;
    }
  }
  // ============================== FOR IN ===================================

  int countComA = 0;
  for (String aluno in alunos) {
    // Quando se tem uma lista para se percorrer, e não nessecita denenhuma variavel de controle
    if (aluno.startsWith("a")) {
      countComA++;
      print(aluno);
    }
  }
  print('${countComA} alunos, começão com A');
  return;
}
